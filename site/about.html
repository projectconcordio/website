Concord is open source consensus engine, used in distributed systems such as Blockchain, to ensure that only valid data is entered into the Blockchain. The core engine of Project Concord is concord-bft, a generic state machine replication library that can handle malicious (byzantine) replicas.

This library is designed to be used as a core building block for replicated distributed data stores, and is especially suited to serve as the basis of permissioned Blockchain systems.
